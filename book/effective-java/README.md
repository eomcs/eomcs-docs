# 이펙티브 자바 3/E(Effective Java, 3rd Edition) 

## 개요

- 자바 프로그래밍 언어의 모범 사례를 다룬 책
- 자바 개발자들에게 권장되는 설계 패턴, 코드 작성 방법, 성능 최적화 기법 등을 소개
- 자바 언어의 기능과 라이브러리를 효과적으로 활용하는 방법을 설명
- 자바 개발자들이 더 나은 코드를 작성하고 유지보수하기 쉽게 만드는 데 도움을 줌
